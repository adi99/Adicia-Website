ADICIA Metaverse

A Voxel based unique virtual world where players can play, own, and monetize content and gaming experiences. A Play To Earn (P2E) game similar to Minecraft or Sandbox but with an in-build Game Maker which will allow the Player to make complex games (puzzle and other genres) without any technical knowledge.

User Generated Game Design and Monetization is main focus of the this platform. Players are incentivised to not only Play-To-Earn (P2E), but also to build Games and Monetize them. We will have A Metaverse game, A Game Maker and NFT Game MarketPlace inside this Platform.

Gamers are our main audience who will earn NFT by playing it, but also creators who will build complex NFT games with templates, scripts and Visual Editors.

Pitch Deck

Hero NFT

Players can Mint the NFTs by exploring the Open World and solving the Puzzle.

These NFTs have HitPoints, AttackDamage and ShieldPoints and can be used during Real Time NFT Battle in Tournament

Inside the Game NFTs can be minted, upgraded and bred. Players can play the Card battle with each other. Winner can earn NFTs and Points, which can be used to upgrade or bred(Combining two NFTs to mint new Powerful NFT).

We need a Marketplace where Creators can buy, sell and trade NFTs, NFTs Games, Virtual Land, Open-world, and in-game Assets. Users can rate and review the NFTs, which will improve purchasing decision process and Sales.

Game Play

Player will land on Metaverse where he can explore the Open-World.

Player reached to the Optical Illusion Puzzle placed inside this World.

Player will mint NFT after solving the Puzzle and move to next world.

Player reached to this next Environment (Medieval Interior) and explore this World.

Player will solve the next Mystery Physics Puzzle.

Player can mint their Second NFT and monetize both on NFT Marketplace.

RoadMap

Phase 01

October 09, 2022

Adding More Levels and Open World

There are only two levels of GamePlay is available to Play. We need at least 20-25 levels of Gameplay to make it engaging for players to play. We also need soundtracks to be included in the Game.

This game needs a bigger Open world which can be explored during Gameplay. Interactive elements can be placed inside the Open world which makes them more immersive.

Phase 02

October 29, 2022

Better Integration of NFT

When Player Solved the Puzzle, they mint a NFT with HitPoints, AttackDamage and ShieldPoints. These NFTs can be Upgraded, bred, and battle with each other inside the Game.Players can also purchase Virtual land and in-game Assets as NFTs and Use them as an assets to build any genre of Games.

Phase 03

November 11, 2022

Real-Time Card Battle System

Inside the Game NFTs can be minted, upgraded and bred. Players can play the Card battle with each other. Winner can earn NFTs and Points, which can be used to upgrade or bred(Combining two NFTs to mint new Powerful NFT).

Phase 04

November 22, 2022

NFT Game MarketPlace

We need a Marketplace where Creators can buy, sell and trade NFTs, NFTs Games, Virtual Land, Open-world, and in-game Assets. Users can rate and review the NFTs, which will improve purchasing decision process and Sales.

Phase 05

December 23, 2022

Voxel Game Maker

The Game-Maker will be a download-able native application (Mac, Windows, Linux). A web application can be created for building Mini Games but to make Big Game Native application needs to be downloaded. So we will have Web version of Game Maker for Mini Game and Desktop Apllication for Big Game. Players can Rate and Review these Games according to their Liking.

Phase 06

December 18, 2022

Metaverse Game Lounch

Full featured Game with A Metaverse game, A Game Maker and NFT Game MarketPlace will be launched. Multiplayer and Tournament will be availble to attract repeat players.
